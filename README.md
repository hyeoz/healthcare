## Description

가상 EMR 데이터를 활용하여 환자 정보와 상세 데이터와 차트를 확인하고 필터링할 수 있는 웹 페이지

- 각 조건에 따른 필터링
- 환자 정보 테이블
  - 테이블 행 개수 설정
  - 테이블 페이지네이션
  - 클릭 시 해당 환자 상세 정보
  - 컬럼 클릭 시 정렬
- 각 조건에 따른 pie 차트

## Development Environment

- OS : Mac OS
- Stack :
  - JavaScript
  - React
  - reactstrap
  - bootstrap
  - Apexcharts
  - Axios

## Usage
~~~
$ git clone https://github.com/hyeoz/healthcare.git
$ yarn
$ yarn start
~~~
